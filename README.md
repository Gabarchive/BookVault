# BookVault

BookVault is a console-based Java application designed to simulate a simple library management system.  
It was developed as a learning project focused on Object-Oriented Programming (OOP), separation of concerns, and basic software architecture using services and entities.

---

## Features

- Register new books  
- Register users  
- Search and list books  
- Basic library management flow  
- Structured using services and entities following OOP principles  

---

## Project Purpose

This project was created to practice and reinforce core Java concepts, including:

- Object-Oriented Programming (classes, objects, encapsulation)  
- Separation of responsibilities (Service layer and Entity layer)  
- Console-based user interaction  
- Basic system organization and architecture  

---

## Technologies Used

- Java (JDK 8+)  
- Console application (Terminal)  
- No external libraries
